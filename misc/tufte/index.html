Tufte in D3

A set of D3.js visualizations to explore Edward Tufte's concept of "data ink". The core of the idea is that we can make visualizations easier to interpret by eliminating visual elements that are redundant or do not encode any data, reducing the clutter and allowing the data to speak for itself.

Visualizations

Each visualization is in a self-contained HTML file - feel free to inspect the page source to see how they were made.

  • Line Chart
  • Bar Chart
  • Box Plot
  • Scatter/Rug Plot
  • Marginal Distribution
  • Sparklines

PROJECTS

Below is a selection of the projects that I am working/have worked on. They include final projects for courses, hackathon projects, side projects, and work for clubs/project teams.

CURRENT PROJECTS

Tython - designing a Python-like language - (2019)

  • language with Python-like syntax and static typechecking
  • Tython code transpiles to Python (goal is to be kind of like Typescript but for Python)
  • currently being rewritten from scratch as of Summer 2019 (see repostory README to track progress!)

PAST PROJECTS

XiC (Xi Compiler) - (2019)

  • compiler for Xi (a strongly-typed, imperative, procedural language) and Xi++ (its object-oriented extension), targeting x64 and implemented in Java
  • developed a dataflow analysis framework and used it to build several optimizations including register allocation, dead code elimination, copy-propagation, and common-subexpressions elimination

12 Hour Club - Shiro - (2019)

  • interactive concept map with dynamically positioned nodes and Prezi-like navigation, implemented in D3

Cornell Data Science - Fake News Challenge - (2018-19)

  • build and visualize models to detect relevance and stance of a news article relative to a claim
  • goal: to increase transparency and interpretability of models and results
  • won a prize from Sandia National Labs at BOOM 2019

DestiNation Matcher - Travel Destination Recommender - (2019)

  • returns destinations based on user's interests and preferences (activities, climate, local language/drinking age, etc)
  • uses custom nicheness metric calculated using network structure of Wikivoyage to emphasize "hidden gems"

OCaMOSS - (2018)

  • source code plagiarism detection software implemented in OCaml, inspired by MOSS

Cornell Data Science - Social Tribes - (2018)

  • unsupervised clustering and visualization of political journalists on Twitter
  • Interactive Demos: agglomerative clustering · dendrograms · force-directed graph

US Foreign Aid - Dynamic Data Visualization - (2017)

  • interactive visualization of US foreign aid flow, implemented in D3

Cornell Cup Robotics - Minibot Platform Team - (2017)

  • software platform for Minibot, Cornell Cup's educational robotics platform
  • helped implement physics backend and optimized frontend display for virtual bot/obstacle simulator
